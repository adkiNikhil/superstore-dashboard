## Project Overview
This interactive dashboard analyzes sales performance and customer purchasing behavior for a global superstore. It utilizes **RFM Analysis (Recency, Frequency, Monetary)** to segment customers and identify high-value targets vs. at-risk churn candidates.

**Live Demo:** [Link to your Streamlit App will go here]

## Key Business Insights
- **Sales Trends:** Analyzed a 4-year dataset to track monthly revenue growth and seasonal spikes.
- **Profitability:** Identified that while the 'Furniture' category drives high revenue, it suffers from the lowest profit margins (-15% in specific regions).
- **Customer Segmentation:** Implemented RFM logic to classify the top 25% of customers as "Champions," allowing for targeted loyalty programs.

## Tech Stack
- **Python:** Core logic and data manipulation.
- **Pandas:** Data cleaning, date parsing, and aggregation.
- **Streamlit:** Interactive web application framework.
- **Plotly:** Dynamic data visualization.
